# 放量推动失败系统

一个专注于：

- 放量推动
- 共识走弱
- 跟随失败
- 加速衰竭
- 情绪高潮

的 Pine Script 交易指标系统。

---

## 核心思想

真正强势的行情：

不仅需要：

- 放量
- 推动

还需要：

- 跟随
- 延续
- 共识增强

如果：

成交量放大，
但单位成交量推动效率下降，

则可能意味着：

- 主力出货
- 推动衰竭
- 跟随失败
- 情绪透支
